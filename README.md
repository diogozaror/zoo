# Zoológico

Essa aplicação foi feita como o segundo trabalho pra matéria de Linguagens de Programação.
O objetivo desse trabalho era conseguir desenvolver uma aplicação usando os conceitos chaves de uma linguagem orientada a objeto.
Nesse trabalho, que fiz em conjunto com meu colega de classe Gabriel Colonese, nós desenvolvemos o que seria um aplicativo para ajudar no controle de um zoológico. O usuário registraria funcionários e animais, e conseguiria ligar cada animal a um funcionário, e por fim conseguiria ver a quantia gasta em salário e comida durante um mês de zoológico funcionando.

A aplicação é bem simples e apenas usa o framework Swing, feito totalmente em java.
